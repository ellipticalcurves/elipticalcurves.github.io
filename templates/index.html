Dash App (click me)

Youtube thumbnail and title data visualisation

Dash app built using the youtube API to retrieve the top youtube thumbnail, do some data analysis on the images (multidimensional scaling) and display/plot results to help mood board video ideas. This was made in python using plotly and dash along with some data analysis algorithms from sci kit learn and tensorflow. I plan to make it better by having better by having more efficient data analysis and better visualisation.

Youtube video - Summer of maths exposition

I made a youtube video for summer of mathematics exposition challenge by 3blue1brown. This video explains an algorithm known as multidimensional scaling which basically makes higher dimensional data fit into 2D or 3D space by looking at the variables that contribute most to the variance. This video was animated using manim, a python library by 3b1b. (Note:The channel name is with 1 L elipticalcurves. )
